# Self-Pruning-NN

This project presents a self-pruning neural network built using PyTorch that learns to eliminate its own unnecessary weights during training, rather than relying on post-processing pruning techniques. The core idea is to associate each weight with a learnable gate parameter, passed through a sigmoid function to constrain it between 0 and 1, and multiplied with the weight to determine its effective contribution. A custom PrunableLinear layer is implemented to support this mechanism, and a multi-layer perceptron is trained on the CIFAR-10 dataset. To encourage sparsity, the training objective combines standard cross-entropy loss with an L1 regularization term applied to all gate values, which drives many of them toward zero and effectively prunes the corresponding connections. Experimental results across different regularization strengths (λ) show a clear trade-off between model accuracy and sparsity, with higher λ producing more compact but less accurate models. The final distribution of gate values demonstrates successful pruning, with a large concentration near zero and a smaller set of active connections, highlighting the model’s ability to automatically identify and retain only the most important features.
